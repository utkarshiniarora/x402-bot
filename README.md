# Deployment Guide
A telegram bot which auto deducts micro-payments on [Solana](https://solana.com/) in stables with [x402](https://www.x402.org/) using [Chainstack](https://chainstack.com/) designed for fast monitoring and clear insights.

---
## Quick Start
**Clone the Repository** 
```bash
git clone https://github.com/utkarshiniarora/x402-bot
cd x402-bot
```
**Install Dependencies**
```bash
npm i
```

**Set up your `.env`** 

For a detailed step-by-step guide with code examples and screenshots, read the full blog: [here]()

**Run the x402 Server**
```bash
node server.js
```

**Run the x402 client**
```bash
node index.js
```
